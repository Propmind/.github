# Security Policy

## Reporting a Vulnerability

Para reportar uma vulnerabilidade de segurança, entre em contato com a equipe de engenharia via e-mail interno antes de abrir uma issue pública.

## Dependency Updates

Todos os repos ativos têm Dependabot configurado com schedule semanal. PRs de segurança são abertas automaticamente — revise e merge com prioridade.

Veja [ADR E-001](https://github.com/Propmind/propmind-dev-cli/blob/main/docs/adr/E-001-dependabot.md) para o racional desta decisão.
